# Fonts Directory

This directory is for storing custom font files (like Söhne from Klim Type Foundry).

## Söhne Font Setup

When you obtain Söhne font files, place them in this directory:

```
src/app/fonts/soehne/
├── Söhne-Buch.woff2          (Regular 400)
├── Söhne-Halbfett.woff2      (Medium 500)
├── Söhne-Kräftig.woff2       (SemiBold 600)
├── Söhne-Breit.woff2         (Bold 700)
├── Söhne-Extrabold.woff2     (ExtraBold 800)
├── Söhne-Mono.woff2          (Mono variant)
└── Söhne-Kursive.woff2       (Kursive variant - optional)
```

**Note:** Currently using Inter (from Google Fonts) as a premium free alternative that provides a similar aesthetic to Söhne.

See `/TYPOGRAPHY.md` in the project root for complete setup instructions.
